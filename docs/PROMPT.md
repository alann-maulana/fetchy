Buatkan PRD untuk aplikasi mobile sederhana untuk manajemen rest api seperti postman:
- Manajemen api client: 
    - Request: http methods, parameters, headers, body, authentication
    - Response: http status code, headers, body
- Manajemen proyek
    - Collections, include fitur import dan export yang compatible dengan Postman Collection v2.1
    - Environment and variables

Manajemen content type untuk api client response:
- Default, tampilkan berupa plain text
- Jika berupa json, tambahkan mode view menggunakan json viewer
- Jika berupa html, tambahkan mode view menggunakan html viewer

Fitur umum :
- Mobile responsive
- Tampilan clean dan support dark mode
- Integrasikan flutter skill berikut https://github.com/flutter/skills

Tech stack :
- Flutter
- Android and iOS
- Dio
- Material 3

Bagi PRD tersebut menjadi 3 fase :
- Skeleton project folder + files utama
- Manajemen api client
- Manajemen proyek
